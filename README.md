# E-Commerce Customer Behavior Analytics Dashboard

This project presents an interactive dashboard built using Microsoft Power BI to analyze customer behavior on an e-commerce platform. The dashboard visualizes important business metrics such as total sessions, total revenue, total purchases, and average time spent on the website.

The dashboard includes multiple visualizations such as traffic source analysis, revenue by country, device usage distribution, and product category purchase trends. These insights help understand how customers interact with the platform and which factors influence purchasing behavior.

This project demonstrates the use of data visualization and business analytics techniques to convert raw e-commerce data into meaningful insights that support better business decisions.
